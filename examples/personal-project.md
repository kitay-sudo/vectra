# Personal project profile

**Goal:** publish a searchable family recipe archive. **Owner/roles:** one person combines Owner, Planner, Writer, and Reviewer; external review is used only for privacy. **Core artifacts:** `PROJECT`, one active `TASK`, `MEMORY`, and decisions for storage and publication.

First loop: inventory ten recipes → define completeness criteria → normalize fields → validate that every source image is represented → review privacy → update the data dictionary. Stop if attribution or consent is unknown. Done means 50 recipes pass schema checks, search tasks succeed, backup restore is tested, and private notes are excluded.

Anti-pattern: documenting every formatting preference as a decision. Tailoring: weekly status and multi-agent artifacts are omitted until coordination exists.
