# Профиль личного проекта

**Цель:** опубликовать доступный для поиска архив семейных рецептов. **Владелец/роли:** один человек совмещает роли Владельца, Планировщика, Автора и Ревьюера; внешняя проверка привлекается только для конфиденциальности. **Ключевые артефакты:** `PROJECT`, одна активная `TASK`, `MEMORY` и решения по хранению и публикации.

Первый цикл: инвентаризировать десять рецептов → определить критерии полноты → нормализовать поля → проверить, что каждое исходное изображение представлено → проверить конфиденциальность → обновить словарь данных. Остановиться, если атрибуция или согласие неизвестны. Готово означает, что 50 рецептов проходят проверки схемы, задачи поиска выполняются успешно, восстановление из резервной копии протестировано, а личные заметки исключены.

Антипаттерн: документировать каждое предпочтение по форматированию как решение. Адаптация: еженедельный статус и мультиагентные артефакты опускаются, пока не возникнет координация.
